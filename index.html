页面

页面由三部分内容组成（内容（结构）、表现、行为）：

  1. 内容（结构）：是我们在页面中可以看到的数据。我们称之为内容。一般内容 我们使用html 技术来展示。
  2. 表现：指的是这些内容在页面上的展示形式。比如说。布局，颜色，大小等等。一般使用CSS 技术实现
  3. 行为：指的是页面中元素与输入设备交互的响应。一般使用 javascript 技术实现。

HTML的初步了解

1. HTML 简介

Hyper Text Markup Language （超文本标记语言）简写HTML，通过标签来标记要显示的网页中的各个部分。网页文件本身是一种文本文件，通过在文本文件中添加标记符，可以告诉浏览器如何显示其中的内容（如：文字如何处理，画面如何安排，图片如何显示等）

2. HTML 文件的书写规范

HTML标签的介绍

1. 标签的格式：

<标签名>封装的数据</标签名>

2. 标签名大小写不敏感

3. 标签拥有自己的属性

（1）基本属性：可以修改简单的样式效果 

（2）时间属性：可以直接设置事件响应后的代码

4、标签的分类

（1）单标签：<标签名 />

（2）双标签：<标签>封装的数据</标签>

5、标签的语法

（1）标签不能交叉嵌套
    例：
    正确：
        <div> <span>哈哈哈</span></div>  
    错误：
        <div><span>哈哈哈</div></span> 

（2）标签必须正确关闭

     1>有文本内容的标签
        例：
         正确：
             <div>哈哈哈</div>
          错误：
              <div>哈哈哈

       2>没有文本内容的标签
        例：
            正确：<br />
             错误：<br>
             
（3）属性必须有值，属性值必须加引号
   例：
    正确：
        <font color=”bule”>哈哈哈</font>
    错误1：
        <font color=bule>哈哈哈</font>
    错误2：
        <font color >哈哈哈</font>

（4）注解不能嵌套
   例：
    正确：
        <!--注解内容-->
    错误：
        <!--<!—注解内容--> -->

注：html代码不是很严谨。有时标签不闭合，也不会报错。

HTML常用标签

1、基本的标签

标签 描述
<!DOCTYPE> 定义文档类型。
<HTML> 定义html
<body> 定义body元素
<h1> to <h6> 定义标题 1 到标题 6。
<p> 定义段落。
<br> 插入一个换行。
<hr> 定义水平线。
<!----> 定义注释。

2、字符格式化

标签 描述
<b> 定义粗体文本。
<font> 不赞成使用。定义文字的字体、尺寸和颜色
<i> 定义斜体文本。
<em> 定义强调文本。
<big> 定义大号字体文本。
<strong> 定义语气更为强烈的强调文本。
<small> 定义小号字体文本。
<sup> 定义上标文本。
<sub> 定义下标文本。
<bdo> 定义文本显示的方向。
<u> 不赞成使用。定义下划线文本。

3、输出

标签 描述
<pre> 定义预格式文本
<code> 定义计算机代码文本。
<tt> 定义打字机文本。
<kbd> 定义键盘文本。
<var> 定义变量。
<dfn> 定义定义项目。
<samp> 定义样本文本。
<xmp> 不赞成使用。定义预格式文本。

4、块级

标签 描述
<acronym> 定义只取首字母的缩写。
<abbr> 定义缩写。
<address> 定义地址元素。
<blockquote> 定义块引用。
<center> 不赞成使用。定义居中字体。
<q> 定义短的引用。
<cite> 定义引用(citation)。
<ins> 定义被插入文本。
<del> 定义被删除文本。
<s> 不赞成使用。定义加删除线的文本。
<strike> 不赞成使用。定义加删除线的文本。

5、链接

标签 描述

<link> 定义资源引用(resource reference)。

6、框架

标签 描述

<noframes> 定义无框架的节

7、输入

标签 描述
<form> 定义表单。
<input> 定义输入域。
<textarea> 定义文本输入区。
<button> 定义按钮。
<select> 定义下拉式菜单。
<optgroup> 定义选项分组。
<option> 定义下拉列表中的可选项。
<label> 为表单元素创建标记。
<fieldset> 将表单内的相关元素分组。
<legend> 定义框架结构的标题。
<isindex> 不赞成使用。定义单行的输入域

8、列表

标签 描述
<ul> 定义无序列表。
<ol> 定义有序列表。
<li> 定义列表项目。
<dir> 不赞成使用。定义目录列表。
<dl> 定义定义列表。
<dt> 定义定义项目。
<dd> 定义定义描述。
<menu> 不赞成使用。定义菜单列表。

9、图像

标签 描述
<img> 定义图像。
<map> 定义图像映射。
<area> 定义图像地图内部的区域。

10、表格

标签 描述
<table> 定义表格

<tr> 定义表格的行
<td> 定义表格单元
<thead> 定义表格的页眉
<tbody> 定义表格的主体
<tfoot> 定义表格的页脚
<col> 定义用于表格列的属性
<colgroup> 定义表格列的组

11、样式

标签 描述
<style> 定义样式定义
<div> 定义文档中的节
<span> 定义文档中的节

12、元信息

标签 描述
<head> 定义关于文档的信息
<title> 定义文档的标题

<base> 定义某个页面当中的所有链接的基准链接
<basefont> 不赞成使用。定义基准字体

13、Programming
